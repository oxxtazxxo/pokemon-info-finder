# Pokémon Info Finder

## Description
Pokémon Info Finder is an interactive web application built with HTML, CSS, and JavaScript. Users can search for any Pokémon by name, browse Pokémon by type, or generate a random Pokémon. The application retrieves live data from the PokéAPI and displays detailed information including the Pokémon's sprite, typing, abilities, height, weight, Pokédex entry, and base stats with visual stat bars.

## How to Run
Download or clone this repository.
Open the project folder.
Launch index.html in a web browser.
Alternatively, use the Live Server extension in Visual Studio Code for the best experience.
Enter a Pokémon name, browse by type, or use the Randomize button.
